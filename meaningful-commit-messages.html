Meaningful commit messages

02 Apr 2018 on Git, Github

A short note, your git commit messages should start with Fix or Add instead of Fixed or Added.
It should fit well in:

When merged/applied, this patch/commit will <commit message>

Example:
When merged, this commit will “fix the photo-viewer crash”

Reference :

Writing Good Commit Messages by Erlang
Git Commit by Chris
